# 🎀 Baza Forms v2 — Briefing & Pedido

Formulário completo de briefing + pedido de produto para clientes da **Baza Personalizados**.

## Funcionalidades

- Navegação em 5 etapas (Cliente → Visual → Produtos → Entrega → Resumo)
- 7 tipos de produtos com campos dinâmicos
- Integração com Formspree (respostas por e-mail)
- Salvar como PDF
- Tela de confirmação após envio

## Como rodar localmente

```bash
npm install
npm run dev
```

Acesse: http://localhost:5173

## Deploy na Vercel

1. Suba no GitHub
2. Importe em vercel.com/new
3. Clique em Deploy
